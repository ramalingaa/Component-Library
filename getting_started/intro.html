Introduction to RAGA UI

RAGA UI is an open source CSS framework designed to fasttrack your application's design processes. RAGA let's you choose different styles for your application from the most popular styles. With the pre-defined class components and utitlities you never have to worry about creating seamless UI for your website. Though RAGA let's you create fluid designs but the functionality is our main purpose.
